Ringkasan Materi



 1. Windows 1.0

   Windows 1.0 dirilis pada 20 November 1985 dan diberi nama “Windows” oleh Rowland Hanson, marketing manager Microsoft Corporation sistem operasi Windows 1.0 adalah sistem operasi berbasis Graphical User InterfacePada versi pertama ini, Microsoft mengintegrasikan banyak program tambahan, seperti Microsoft Paintbrush, Write, Notepad, Calendar, Calculator, CardFile, dan juga Clipboard

 2. Windows 2.0

   Windows 2.0 dirilis pada 9 Desember 1987, Windows 2.0 dapat menjalankan aplikasi secara multitasking (bersamaan).Windows 2.0 mendukung penggunaan kartu grafis dengan spesifikasi VGA (Video Graphics Array) sehingga dapat menampilkan resolusi 640 x 480 pada kedalaman warna 4bit. Windows 2.0 mampu mengakses memori sampai dengan 1MB. Windows 2.0 sudah mengunakan procesor intel 80268.

 3. Windows 3.0

   Windows 3.0 dirilis pada tahun 1990, Windows 3.0 merupakan perbaikan yang signifikan dari versi sebelumnya. Sebelum adanya Sistem Operasi Microsoft Windows yang berbasis antarmuka grafis, sudah ada MS-DOS (Microsoft Disk Operating System) yang dirilis pada tahun 1981. Windows 3.0 dapat berjalan di dalam tiga modus, yakni modus real, modus standar, dan modus 386 Enhanced, dan kompatibel dengan prosesor-prosesor keluarga Intel dari Intel 8086/8088,80286, hingga 80386.

 4. Windows NT 3.1

   Produk ini mulai diproduksi tanggal 27 Juli 1993. Windows NT 3.1 dapat berjalan pada CPU Intel x86, DEC Alpha, and MIPS R4000. Windows NT tersedia dalam 2 versi yaitu : Windows NT 3.1 dan Windows Advance Server. Windows ini merupakan windows mengunakan sistem operasi 32-bit. Didesain agar kompatibel dengan Sistem Operasi terdahulu seperti MS Dos, IBM OS/2, windows NT dibangun dari sistem pengembangan dari Sistem Operasi IBM OS/2 dalam pengembangan sistem operasi tersebut Microsoft dan IBM melakukan perkembangan secara bersama sama.
