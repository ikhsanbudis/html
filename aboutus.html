Apa Itu Website ?


    Website merupakan istilah yang cukup familiar. Kini, banyak orang tertarik membuat website karena berbagai manfaat yang diberikan.atau bisa juga di artikan sebuah kumpulan halaman pada suatu domain di internet yang dibuat dengan tujuan tertentu dan saling berhubungan serta dapat diakses secara luas melalui halaman depan (home page) menggunakan sebuah browser menggunakan URL website.

Tujuan Membuat Website


     Tujuan saya membuat website ini adalah supaya saya bisa berbagi informasi kepada semua pembacanya. Dan supaya orang yang membaca webite saya ini bisa mendapat ilmu yang bermanfaat bagi mereka. Dengan adanya web ini saya harapkan bisa menambah wawasan bagi semua orang dan bisa untuk media pembelajaran baik dirumah, disekolahan maupun ditemapt umum.

History


    Awal mula saya mendirikan website ini adalah karena saya ingin menambah wawasan saya. Dan saya ingin
dalam web programer karena itu menjadi bekal saya besok untuk mencari pekerjaan yang mapan.
Oleh karena itu saya termotivasi untuk membangun website ini dan semoga website ini dapat memberikan informasi yang bermanfaat bagi pembaca. website ini didirikan pada tahun 2019.

Location


    Lokasi pusat web ini adalah berada di yogyakarta, lebih tepatnya ada di sleman. Jika ada informasi yang kurang mohon untuk konfirmasi via email kami, email kami sudah saya sediakan di bagian bawah web ini. atau bisa kontak via whatapp kami yang sudah disediakan dibagian bawah website.
